### 요약
* **stage area** : 
* **git init** : 현재 디렉토리를 root 디렉토리로 하는 프로젝트의 버전관리 디렉토리(.git)를 생성
* **git add [filename]** : 프로젝트의 현재 상태를 .git 파일에 저장
* **git status** : 현재 프로젝트의 상태와 .git에 저장된 프로젝트의 상태를 비교
